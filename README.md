# Veni Vici Dog Scout

Veni Vici Dog Scout is a React web app that helps users discover random dog breeds one at a time by pulling data from The Dog API. Each result shows a matching dog image along with consistent breed details such as breed name, breed group, life span, and temperament. Users can click a breed name to add it to a ban list, which prevents that breed from appearing in future results, and the app also keeps a session history of previously viewed dogs.
